Hugo 使用记录

安装 Hugo

brew install hugo
hugo version

创建一个网站项目

hogo new site quickstart

使用一个网站主题

cd quickstart
git init
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke
# 修改文件使用主题
echo 'theme = "ananka"' >> config.toml

添加文章文件

hugo new posts/my-first-post.md

运行

hugo server -D

浏览地址：http://localhost:1313
